# InvoiceGST — Free GST Invoice Generator & Calculator

> A free, fully functional GST Invoice Generator built for Indian freelancers and businesses.

🔗 **Live Demo**: [Deploy on Vercel — see below]  
📦 **Built with**: React 18 + Vite + Vanilla CSS  
💸 **Cost**: ₹0 — no paid subscriptions, no backend, no sign-up

---

## Features

- ✅ **GST Calculator** — CGST + SGST (intra-state) or IGST (inter-state)
- ✅ **5 GST Slabs** — 0%, 5%, 12%, 18%, 28%
- ✅ **Invoice Builder** — Seller, buyer details, line items with HSN/SAC codes
- ✅ **Live Invoice Preview** — See changes in real-time
- ✅ **PDF Export** — Print-to-PDF with clean print styles
- ✅ **Quick GST Calculator** — Standalone instant calculator widget
- ✅ **GST Rate Guide** — Know which slab applies to you
- ✅ **GST Filing Tips** — Stay compliant
- ✅ **Fully Responsive** — Works on all screen sizes
- ✅ **Dark Mode** — Premium dark UI with bento grid layout

---

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

---

## Deploy to Vercel (Free)

1. Push this repo to GitHub
2. Go to [vercel.com](https://vercel.com) → New Project → Import your repo
3. Framework: **Vite** (auto-detected)
4. Click **Deploy** → Done!
